## 🚀 DEPLOYMENT IN 20 MINUTES

### Step 1: Prepare (5 min)
```
1. Create Hugging Face account (free): huggingface.co
2. Gather credentials:
   - SUPABASE_URL
   - SUPABASE_SERVICE_ROLE_KEY
   - JWT_SECRET
   - TERABOX username/password
```

### Step 2: Create Space (2 min)
```
1. Go to: huggingface.co/new-space
2. Fill form:
   - Name: arsip-anka
   - SDK: Docker
   - Visibility: Public
3. Click: Create Space
```

### Step 3: Link GitHub (1 min)
```
1. Space Settings → Repository tab
2. Link: https://github.com/donyp/arsipankabaru.git
3. Auto-deploy on push!
```

### Step 4: Add Secrets (2 min)
```
1. Space Settings → Secrets tab
2. Add 6 environment variables:
   SUPABASE_URL
   SUPABASE_SERVICE_ROLE_KEY
   JWT_SECRET
   TERABOX_WEBDAV_URL
   TERABOX_USER
   TERABOX_PASS
```

### Step 5: Wait & Test (10 min)
```
1. Watch build (⏳ 3-5 minutes)
2. Once green (✅), click app URL
3. Login and test file upload
4. Verify Terabox sync
5. Done! 🎉
```

**Total: 20 minutes to production!**

### Hugging Face Spaces Features
- ✅ Completely FREE (no credit card needed)
- ✅ 50GB persistent storage
- ✅ Public URL provided
- ✅ Auto-deploy from GitHub
- ✅ Docker support

### Limitations (Good to Know)
- ⚠️ Sleeps after 48 hours idle (auto-wakes on request)
- ⚠️ Cold start 5-10 seconds on first request
- ⚠️ Shared CPU (might be slower than VPS)
- ⚠️ Can restart without warning (rare)

### Performance
- Expected: Good for team testing
- Not suitable for: High-traffic production
- Can upgrade to: VPS after testing
